# gamno v0.3 Power Demo

Gamno — демо-инструмент для ежедневного чек-ина и анализа решений в desktop-формате.

## Что это демо
- Дашборд: сводка 7 дней, сигналы, прогноз индекса.
- История: плотная таблица, детали записи, шаблон для нового чек-ина.
- Оракул: сценарии what-if с картой влияний.
- Граф влияний: редактируемая матрица причинности.
- Настройки (Data Vault): экспорт/импорт/очистка/генерация демо-данных.

## Где хранятся данные
Все данные хранятся локально в браузере (IndexedDB, Dexie).

## Экспорт/импорт
- **Экспорт данных** — выгружает локальную БД в файл `gamno-backup-YYYY-MM-DD.json`.
- **Импорт данных** — восстанавливает БД из файла после подтверждения.
- **Очистить данные** — удаляет локальные данные.

## Демо-данные
В настройках есть кнопка **Сгенерировать демо-данные (30 дней)**.
Можно указать seed для детерминированной генерации.
